# 飞享 - 无人机服务平台小程序

## 项目简介
飞享是一个无人机服务平台微信小程序，旨在为用户提供无人机服务需求发布、服务采购、社区交流等功能。

## 项目结构
```
├── pages              // 页面文件夹
│   ├── index          // 欢迎页
│   ├── login          // 登录页面
│   ├── home           // 首页
│   ├── demand         // 需求发布
│   ├── service        // 服务采购
│   ├── community      // 社区交流
│   └── my             // 个人中心
├── static             // 静态资源
│   ├── tabbar         // 底部导航图标
│   ├── icon           // 图标
│   └── ...            // 其他静态资源
├── utils              // 工具函数
├── app.js             // 小程序入口文件
├── app.json           // 小程序配置文件
├── app.wxss           // 小程序全局样式
└── project.config.json // 项目配置文件
```

## 功能模块
1. **欢迎页**: 展示平台介绍，引导用户登录或进入首页
2. **登录/注册**: 用户身份验证
3. **首页**: 展示平台主要服务和内容
4. **需求发布**: 用户发布无人机服务需求
5. **服务采购**: 浏览和购买无人机相关服务
6. **社区交流**: 用户之间交流、分享经验和资讯
7. **个人中心**: 管理个人信息、订单、收藏等

## 开发环境
- 微信开发者工具
- Node.js环境

## 运行说明
1. 克隆本项目到本地
2. 打开微信开发者工具
3. 导入项目
4. 在project.config.json中配置您的AppID

## 备注
本项目是由uni-app框架转换为原生微信小程序开发框架的版本。
